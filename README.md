# dashboard-secret


## Установка

```bash
$ yarn
```
Ensure also that [Docker is installed](https://docs.docker.com/engine/install) on your work station


## Запускать приложение используя node

```bash
# development
$ yarn start:dev
or
nest start

# Debug/watch
$ yarn start:debug

# production
$ yarn build:prod
$ yarn start
```
